# Yarn scripts build tool boilerplate

Contains the following tasks:

## Dev tasks

* SCSS compilation
* SCSS linting
* SCSS sourcemaps
* CSS autoprefixing
* ES6 support
* JS sourcemaps
* Image minification
* SVG icon sprites
* File watching

## Build tasks

* SCSS compilation
* CSS autoprefixing
* CSS minification
* ES6 support
* JS minification
* Image minification
* SVG icon sprites
